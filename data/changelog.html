ContextMenuPlus Changelog

Version 1.1
  • Add-on SDK upgrade.
  • Added recovery tool for recovering preferences(because of a bug in Add-on SDK 1.4).[No actions required from users]
Version 1.0
  • Now uses simple-prefs module for preferences. Preferences/Options in Add-ons Manager replaced Options panel.
  • Added new items: Stop Loading Image, Copy Feed URL.
Version 0.8
  • Now trims produced strings of "Copy as HTML Link(Selection)", "Copy as Link(Selection)" and "Copy Text and URL".
  • Fixed a bug.
Version 0.7
  • Added new item: Copy Text and URL
  • Set a width for shortcut key inputs.
Version 0.6
  • Added new items: Copy Page Title and URL, Copy Title & URL of all Tabs, Copy as HTML Link (Page), Copy as Link (Page), Copy as HTML, Copy as HTML Link(Selection), Copy as Link(Selection).
  • Changed layout of Options panel.
Version 0.5
  • Fixed text color issue in Options panel on Mac.
  • Now checks shortcut keys confliction.
  • Changed style of notification message.
  • Fixed some other bugs.
Version 0.4
  • New context menu item: Copy Page URL
  • Added shortcuts for "Copy as Plain Text", "Copy Page Title" and "Copy Page URL".
  • Added "ContextMenuPlus Options"(accessible from Add-on Bar). In Options panel everything can be disabled or enabled and shortcut keys can be changed.
  • Shows changelog page after upgrade.
Version 0.3
  • Fixed "Open Page Hostname" giving error on some pages like about:blank, about:home, ...
Version 0.2
  • Now trims whitespace characters from both sides of "Copy as Plain Text" and "Copy Link Text" produced strings.
  • Firefox 7 compatibility
Version 0.1
  • First version of ContextMenuPlus
